#Linkdood开发指南

##功能概述
linkdood是北信源软件股份有限公司为移动应用提供的完善即时通讯开发框架，IM基础功能满足App用户需求的基本功能

从聊天方式上包括单聊、群聊、聊天室；从信息类型上包括文字、表情、图片、语音、视频、地理位置、通知消息。通过豆豆平台，开发者不必搭建服务端硬件环境，就可以将即时通讯、实时网络能力快速集成至应用中。
